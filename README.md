# Scalable e-commerce
This project comprises a few microservices, each with it's own codebase, connected by gRPC protocols.

# Technology stack
 - gRPC
 - sqlite3

# Used packages
 - `cleanenv` for reading config
 - `slog` for logging
 - `golang-migrate` for database migrations
